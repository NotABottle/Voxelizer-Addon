# Voxelizer-Addon
An addon for blender that allows voxelization through the press of a button with support for textured and skinned meshes.

### Showcase Footage
https://github.com/user-attachments/assets/a118982e-52f5-4fbd-b97b-017580d747fb

### Description
Blender addon that allows the user to select a mesh, press the "Voxelize" button, and create a duplicate voxelized version of the original mesh.
