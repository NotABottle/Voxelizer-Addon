# Voxelizer-Addon
Voxelizer is a Blender addon for creating voxel-based assets easily within your scenes.

### Showcase Footage
https://github.com/user-attachments/assets/a118982e-52f5-4fbd-b97b-017580d747fb

### Description
A Blender addon that allows the user to select a mesh, press the "Voxelize" button, and create a duplicate voxelized version of the original mesh.

### How to use
1. Select either a rigged or static mesh.
2. Tick the box corresponding to the type of mesh you are attempting to voxelize.
3. Click Voxelize.
4. New collections will be automatically created for organizational purposes.
